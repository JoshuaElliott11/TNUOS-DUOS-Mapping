Network Tariffs in GB
This page covers the main network tariffs in GB and how to use them in Gridcog
This section explains the two major network charges you’ll see in the UK electricity
market: Distribution Use of System charges (DUoS) and Transmission Network Use of
System charges (TNUoS). These are non-commodity (non-energy) costs that help
recover the cost of building, operating and maintaining the distribution and transmission
electricity networks respectively.
The table below provides an indicative view of which tariffs apply to different site types.
Note that rules may change over time, so it’s recommended to verify against actual bills
where possible.
M ore information on each of these tariffs is provided below.

=== PAGE 16 ===
16
Under today’s methodology, each generation zone has four tariff elements, which
combine differently depending on generator type. The wider tariff is made up of:
1.Peak Security Tariff
2.Shared Year-Round Tariff
3.Not-Shared Year-Round Tariff
4.Adjustment Tariff
Two of these may be multiplied by the generator’s Annual Load Factor (ALF), depending
on whether the asset is intermittent or conventional.
The formula for the Wider Generation is as follows 
Conventional Carbon Generators
(e.g. Biomass, CHP, Coal, Gas, Pumped Storage, Battery)
Peak Element + [ Year Round Shared Element x ALF ] + [Year Round Not Shared Element x
ALF] + Adjustment Tariff
Conventional Low Carbon Generators
(for example: Hydro, Nuclear)
Peak Element + [ Year Round Shared Element x ALF ] + Year Round Not Shared Element  +
Adjustment Tariff
Intermittent Generators
(for example: Wind, Wave, Tidal, Solar)
[ Year Round Shared Element x ALF ] + Year Round Not Shared Element + Adjustment
Tariff
ALFs are used to scale the Shared Year-Round element of tariffs for each generator, and
the Year Round Not Shared for Conventional Carbon generators, so that each has a tariff
appropriate to its historical load factor.
